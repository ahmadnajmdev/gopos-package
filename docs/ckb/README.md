# بەڵگەنامەی GoPOS

بەخێربێیت بۆ بەڵگەنامەی GoPOS. سیستەمی GoPOS سیستەمێکی گشتگیری خاڵی فرۆشتن (POS) و بەڕێوەبردنی کارەکانە کە بۆ بازاڕی عێراق/کوردستان دیزاین کراوە لەگەڵ پشتگیری سێ زمان (ئینگلیزی، عەرەبی، کوردی سۆرانی).

---

## GoPOS چییە؟

GoPOS چارەسەرێکی تەواوە بۆ بەڕێوەبردنی کار کە تێکەڵ دەکات:
- **خاڵی فرۆشتن (POS)** - پرۆسەکردنی فرۆشتنی خێرا و کاریگەر
- **بەڕێوەبردنی کۆگا** - کۆنترۆڵی تەواو بەسەر کۆگا لە شوێنی جیاواز
- **ژمێریاری** - بەڕێوەبردنی دارایی تەواو و ڕاپۆرتەکان
- **سەرچاوە مرۆییەکان (HR)** - بەڕێوەبردنی کارمەندان، ئامادەبوون، و مووچە

### سوودە سەرەکییەکان

| سوود | وەسف |
|------|------|
| **سێ زمانە** | پشتگیری تەواو بۆ ئینگلیزی و عەرەبی و کوردی سۆرانی |
| **تەواو** | پێویست ناکات سیستەمی زۆر - هەموو شت یەکخراوە |
| **ڕووکاری مۆدێرن** | ڕووکارێکی پاک و ئاسان لەگەڵ FilamentPHP |
| **نەرم** | دەسەڵاتەکان بەپێی ڕۆڵ دەگونجێنرێن بۆ هەر پێکهاتەیەکی کار |
| **گەورەبوون** | لە فرۆشگایەکەوە بۆ کارەکانی چەند کۆگایەک |

## سیناریۆ باوەکانی کار

### "فرۆشگایەکی تاکە بەڕێوە دەبەم"

**سەرەکی ئەمانە بەکاردەهێنیت:**
1. **مۆدیولی POS** - فرۆشتنی ڕۆژانە، بەڕێوەبردنی شەفت، وەفاداری کڕیار
2. **مۆدیولی کۆگا** - بەدواداچوونی کۆگا، ئاگاداری داواکردنەوە (Reorder)
3. **RBAC** - دانانی ڕۆڵی کاشێر و بەڕێوەبەر

**دەستپێبکە بە:** [QUICK-START.md](QUICK-START.md) → [ڕێنمایی POS](02-pos-enhancements.md)

---

### "چەند کۆگام هەیە"

**سەرەکی ئەمانە بەکاردەهێنیت:**
1. **مۆدیولی کۆگا** - چەند کۆگا، گواستنەوەی کۆگا، بەدواداچوونی وەجبە (Batch)
2. **مۆدیولی POS** - فرۆشتن لە هەر شوێنێک
3. **ژمێریاری** - بەدواداچوونی تێچوو لە شوێنەکان (ناوەندی تێچوو)

**دەستپێبکە بە:** [ڕێنمایی کۆگا](03-inventory-enhancements.md)

---

### "پێویستم بە ڕاپۆرتی داراییە"

**سەرەکی ئەمانە بەکاردەهێنیت:**
1. **مۆدیولی ژمێریاری** - تۆمارەکانی ڕۆژنامە، ساغکردنەوەی بانک، بودجە
2. **ڕاپۆرتەکان** - لیستەکانی دارایی، تەرازووی تاقیکردنەوە

**دەستپێبکە بە:** [ڕێنمایی ژمێریاری](04-accounting-enhancements.md)

---

### "کارمەند بەڕێوە دەبەم"

**سەرەکی ئەمانە بەکاردەهێنیت:**
1. **مۆدیولی HR** - دۆسیەی کارمەند، ئامادەبوون، بەڕێوەبردنی مۆڵەت
2. **مووچە** - ئامادەکردنی مووچە، پسوڵەی مووچە
3. **RBAC** - دەسەڵاتی دەستگەیشتنی کارمەند

**دەستپێبکە بە:** [ڕێنمایی HR](05-hr-module.md)

## پێکهاتەی بەڵگەنامە

بەڵگەنامەی هەر مۆدیولێک پێکهاتەیەکی یەکسان دەگرێتەبەر:

1. **پوختە** - ئەم مۆدیولە چییە و بۆچی بەکاری دەهێنیت
2. **سوودە سەرەکییەکان** - چی کێشەیەک چارەسەر دەکات
3. **حاڵەتەکانی بەکارهێنان** - سیناریۆی دونیای ڕاستەقینە
4. **تایبەتمەندییەکان** - وردەکاری تایبەتمەندییەکان لەگەڵ:
   - چی دەکات
   - کەی بەکاری دەهێنیت
   - چۆن بەکاری دەهێنیت (ڕووکار + کۆد)
   - نموونەی کارەکان
5. **سەرچاوەی تەکنیکی** - API، نەخشەی داتابەیس، نموونەی کۆد
6. **چارەسەرکردنی کێشە** - کێشە باوەکان و چارەسەرەکانیان
