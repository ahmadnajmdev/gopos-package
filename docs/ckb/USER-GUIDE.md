# ڕێنمایی بەکارهێنەری GoPOS

بەخێربێیت بۆ **GoPOS** - سیستەمێکی تەواو بۆ خاڵی فرۆشتن (POS) و بەڕێوەبردنی بازرگانی کە بۆ فرۆشگاکانی عێراق و کوردستان دیزاین کراوە. ئەم ڕێنماییە یارمەتیت دەدات بۆ تێگەیشتن و بەکارهێنانی هەموو تایبەتمەندییەکانی سیستەم بەشێوەیەکی کارا.

## دەستپێکردن

### چوونەژوورەوە

1. وێبگەڕەکەت بکەرەوە و بڕۆ بۆ ناونیشانی GoPOS
2. **ناونیشانی ئیمەیڵ** و **وشەی نهێنی** بنووسە
3. کلیک لەسەر **چوونەژوورەوە** بکە

### ڕێکخستنی یەکەم جار

دوای چوونەژوورەوە بۆ یەکەم جار، پێشنیار دەکەین:

1. **پڕۆفایلەکەت نوێ بکەرەوە** - کلیک لەسەر ناوەکەت لە گۆشەی سەرەوەی چەپ/ڕاست
2. **زمانەکەت دیاری بکە** - هەڵبژێرە لەنێوان ئینگلیزی، عەرەبی، یان کوردی
3. **ئاشنا ببە بە مێنووی لای** - گەڕان لەنێوان بەشە جیاوازەکان

### تێگەیشتن لە ڕۆڵەکەت

دەستگەیشتنت بە تایبەتمەندییەکان پەیوەستە بە ڕۆڵی دیاریکراوت:

| ڕۆڵ | ئاستی دەستگەیشتن |
|------|--------------|
| **سەرپەرشتیاری سیستەم** | دەستگەیشتنی تەواو بە هەموو تایبەتمەندییەکان |
| **بەڕێوەبەر** | POS، فرۆشتن، کۆگا، ڕاپۆرتەکان |
| **ژمێریار** | فرۆشتن، کڕین، ژمێریاری تەواو، مووچە |
| **کاشێر** | کارەکانی POS تەنها |
| **کارمەندی کۆگا** | بەڕێوەبردنی کۆگا |
| **بەڕێوەبەری کارمەندان** | دەستگەیشتنی تەواو بە کارمەندان |
| **کارمەندی HR** | بەڕێوەبردنی ئامادەبوون و مۆڵەت |

## پوختەی داشبۆرد

داشبۆرد پوختەیەکی خێرا لە بازرگانیەکەت پێشکەش دەکات:

- **فرۆشتنی ئەمڕۆ** - کۆی فرۆشتن بۆ ئەمڕۆ
- **داهاتی مانگانە** - داهات بۆ ئەم مانگە
- **ئاگادارییەکانی کەمبوونی کۆگا** - بەرهەمەکان کە پێویستیان بە داواکردنەوە هەیە
- **مامەڵە دوایییەکان** - دوایین فرۆشتن و کڕینەکان
- **بەرهەمە زۆر فرۆشراوەکان** - بەرهەمە باشترینەکان
- **کردارە خێراکان** - قەدبڕەکان بۆ کارە باوەکان

## خاڵی فرۆشتن (POS)

بەشی POS بۆ مامەڵەی خێرا و کارا دیزاین کراوە.

### دەستپێکردنی شەفت (Shift)

پێش فرۆشتن، دەبێت شەفتێک بکەیتەوە:

1. بڕۆ بۆ **POS** > **بەڕێوەبردنی شەفت**
2. کلیک لەسەر **کردنەوەی شەفت** بکە
3. **بڕی پارەی دەستپێکردن** لە سندووقەکەت بنووسە
4. کلیک لەسەر **دڵنیاکردنەوە** بکە

### کردنی فرۆشتن

1. بڕۆ بۆ **POS** لە مێنووی سەرەکی
2. **بەرهەم زیاد بکە:**
   - بارکۆد سکان بکە، یان
   - بە ناوی بەرهەم بگەڕێ، یان
   - کلیک لەسەر بەرهەم لە لیستەکە بکە
3. **بڕ ڕێکبخە:** دوگمەکانی +/- بەکار بهێنە یان ڕاستەوخۆ بنووسە
4. **کڕیار هەڵبژێرە** (ئارەزوومەندانە): بگەڕێ و کڕیارێک هەڵبژێرە
5. **داشکاندن جێبەجێ بکە** (ئەگەر هەبوو)
6. کلیک لەسەر **پارەدان** بکە

### پرۆسێسکردنی پارەدان

GoPOS چەندین شێوازی پارەدان پشتگیری دەکات:

#### پارەدانی تاک
1. شێوازی پارەدان هەڵبژێرە (نەقد، کارت، گواستنەوەی بانکی)
2. بڕی وەرگیراو بنووسە
3. سیستەم بە شێوەی خۆکار گەڕاوە (Baqi) حیساب دەکات
4. کلیک لەسەر **تەواوکردنی فرۆشتن** بکە

#### دابەشکردنی پارەدان (Split Payment)
بۆ کڕیارانێک کە بە چەندین شێواز پارە دەدەن:

1. کلیک لەسەر **دابەشکردنی پارەدان** بکە
2. بڕ بۆ شێوازی یەکەم بنووسە
3. کلیک لەسەر **زیادکردنی پارەدان** بکە
4. شێوازی دووەم هەڵبژێرە و بڕ بنووسە
5. دووبارە بکەرەوە تا بڕی تەواو دابپۆشرێت
6. کلیک لەسەر **تەواوکردنی فرۆشتن** بکە

### هەڵگرتن و بیرهێنانەوەی فرۆشتن (Hold & Retrieve)

**بۆ هەڵگرتنی فرۆشتن:**
1. لە کاتی پارەدان، کلیک لەسەر **هەڵگرتنی فرۆشتن** بکە
2. تێبینی سەرچاوە بنووسە (وەک ناوی کڕیار)
3. فرۆشتن بۆ ٢٤ کاتژمێر پاشەکەوت دەکرێت

**بۆ بیرهێنانەوەی فرۆشتنی هەڵگیراو:**
1. کلیک لەسەر دوگمەی **فرۆشتنە هەڵگیراوەکان** بکە
2. فرۆشتن هەڵبژێرە بۆ بیرهێنانەوەی
3. بەردەوام بە لەگەڵ پارەدان

### بەرنامەی وەفاداری کڕیار (Loyalty)

ئەگەر بەرنامەی وەفاداری چالاک بوو، کڕیاران خاڵ دەبەنەوە لەسەر کڕین:

- خاڵەکان بە شێوەی خۆکار بەپێی بڕی کڕین دەدرێن
- باڵانسی خاڵەکانی کڕیار لە کاتی پارەدان ببینە
- خاڵەکان بگۆڕە بۆ داشکاندن

### داخستنی شەفت

لە کۆتایی شەفتەکەت:

1. بڕۆ بۆ **POS** > **بەڕێوەبردنی شەفت**
2. کلیک لەسەر **داخستنی شەفت** بکە
3. سندووقەکەت بژمێرە
4. **بڕی ژمێردراو** بنووسە
5. پوختەی شەفت ببینەوە
6. کلیک لەسەر **دڵنیاکردنەوەی داخستن** بکە

### بینینی ڕاپۆرتی شەفت (ڕاپۆرتی Z)

1. بڕۆ بۆ **POS** > **ڕاپۆرتی شەفت**
2. شەفت هەڵبژێرە بۆ بینین
3. ببینە:
   - کۆی فرۆشتن
   - وردەکاری شێوازی پارەدان
   - جوڵەی پارە (هاتوو/دەرچوو)
   - جیاوازی (چاوەڕوانکراو بەرامبەر ژمێردراو)
4. کلیک لەسەر **چاپ** بکە بۆ دروستکردنی ڕاپۆرت

## بەڕێوەبردنی فرۆشتن

### دروستکردنی فرۆشتن (لە ئۆفیس)

بۆ فرۆشتنانێک کە لە POS پرۆسێس نەکراون:

1. بڕۆ بۆ **فرۆشتن** > **فرۆشتن**
2. کلیک لەسەر **دروستکردنی فرۆشتن** بکە
3. **کڕیار** هەڵبژێرە (یان بەتاڵی بهێڵەرەوە بۆ کڕیاری ڕاستەوخۆ)
4. بەرهەمەکان زیاد بکە
5. هەر **داشکاندنێک** زیاد بکە
6. کلیک لەسەر **پاشەکەوت** بکە

### بینینی فرۆشتنەکان

1. بڕۆ بۆ **فرۆشتن** > **فرۆشتن**
2. فلتەرەکان بەکار بهێنە بۆ دۆزینەوەی فرۆشتنی دیاریکراو:
   - ماوەی بەروار
   - کڕیار
   - دۆخ
   - دۆخی پارەدان

### چاپی پسوڵە

1. تۆماری فرۆشتن بکەرەوە
2. کلیک لەسەر **چاپی پسوڵە** بکە
3. **چاپ** یان **دابەزاندن** (Download) هەڵبژێرە

### پرۆسێسکردنی گەڕاندنەوە (Returns)

كاتێک کڕیارێک بەرهەم دەگەڕێنێتەوە:

1. بڕۆ بۆ **فرۆشتن** > **گەڕاندنەوەی فرۆشتن**
2. کلیک لەسەر **دروستکردنی گەڕاندنەوە** بکە
3. **فرۆشتنی ڕەسەن** هەڵبژێرە
4. بەرهەمەکان هەڵبژێرە کە دەگەڕێنرێنەوە
5. هۆکاری گەڕاندنەوە بنووسە
6. شێوازی گەڕاندنەوەی پارە هەڵبژێرە
7. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی کڕیارەکان

#### زیادکردنی کڕیار
1. بڕۆ بۆ **فرۆشتن** > **کڕیارەکان**
2. کلیک لەسەر **دروستکردنی کڕیار** بکە
3. وردەکارییەکان پڕ بکەرەوە: ناو، تەلەفۆن، ئیمەیڵ، ناونیشان
4. کلیک لەسەر **پاشەکەوت** بکە

#### بینینی لیستی حسابی کڕیار
1. تۆماری کڕیار بکەرەوە
2. کلیک لەسەر **لیستی حساب** بکە
3. هەموو مامەڵەکان و باڵانس ببینە
4. کلیک لەسەر **چاپ** بکە بۆ دروستکردنی لیست

## بەڕێوەبردنی کڕین

### دروستکردنی داواکاری کڕین

1. بڕۆ بۆ **کڕین** > **کڕینەکان**
2. کلیک لەسەر **دروستکردنی کڕین** بکە
3. **دابینکەر** هەڵبژێرە
4. بەرهەمەکان بە بڕ و نرخی تێچوو زیاد بکە
5. **ژمارەی پسوڵە**ی دابینکەر بنووسە
6. کلیک لەسەر **پاشەکەوت** بکە

### پرۆسێسکردنی گەڕاندنەوەی کڕین

بۆ گەڕاندنەوەی بەرهەم بۆ دابینکەران:

1. بڕۆ بۆ **کڕین** > **گەڕاندنەوەی کڕین**
2. کلیک لەسەر **دروستکردنی گەڕاندنەوە** بکە
3. **کڕینی ڕەسەن** هەڵبژێرە
4. بەرهەمەکان هەڵبژێرە بۆ گەڕاندنەوە
5. هۆکار بنووسە
6. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی دابینکەران

1. بڕۆ بۆ **کڕین** > **دابینکەران**
2. کلیک لەسەر **دروستکردنی دابینکەر** بکە
3. پڕ بکەرەوە: ناوی کۆمپانیا، کەسی پەیوەندی، تەلەفۆن، مەرجەکانی پارەدان
4. کلیک لەسەر **پاشەکەوت** بکە

## بەڕێوەبردنی کۆگا

### بەڕێوەبردنی بەرهەم

#### دروستکردنی بەرهەم

1. بڕۆ بۆ **کۆگا** > **بەرهەمەکان**
2. کلیک لەسەر **دروستکردنی بەرهەم** بکە
3. **زانیاری سەرەکی** پڕ بکەرەوە:
   - ناو (پێویستە)
   - کۆدی بەرهەم/SKU
   - بارکۆد
   - پۆل (Category)
   - یەکەی پێوانە
4. **نرخدانان** دیاری بکە:
   - نرخی تێچوو
   - نرخی فرۆشتن
5. **کۆگا** ڕێکبخە:
   - بەدواداچوونی کۆگا (بەڵێ/نەخێر)
   - ئاستی ئاگاداری کەمبوونی کۆگا (Reorder Level)
   - چالاککردنی بەدواداچوونی وەجبە (Batch) (بۆ بەرواری بەسەرچوون)
   - چالاککردنی بەدواداچوونی ژمارەی زنجیرەیی (Serial) (بۆ تاکە یەکەکان)
6. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی چەند کۆگا

#### ڕێکخستنی کۆگاکان

1. بڕۆ بۆ **کۆگا** > **کۆگاکان**
2. کلیک لەسەر **دروستکردنی کۆگا** بکە
3. بنووسە: ناوی کۆگا، ناونیشان، کەسی پەیوەندی
4. کلیک لەسەر **پاشەکەوت** بکە

### گواستنەوەی کۆگا

بۆ گواستنەوەی کۆگا لەنێوان کۆگاکان:

1. بڕۆ بۆ **کۆگا** > **گواستنەوەی کۆگا**
2. کلیک لەسەر **دروستکردنی گواستنەوە** بکە
3. کۆگای سەرچاوە و مەبەست هەڵبژێرە
4. بەرهەم و بڕ زیاد بکە
5. کلیک لەسەر **پاشەکەوت** بکە (ڕەشنووس دروست دەکات)
6. کلیک لەسەر **ناردن** بکە بۆ پەسەندکردن
7. پەسەندکەر پێداچوونەوە دەکات و کلیک لەسەر **پەسەندکردن** دەکات
8. کلیک لەسەر **تەواوکردن** (Receive) بکە کاتێک بەرهەمەکان وەردەگیرێن

### ژماردنی کۆگا (Inventory Count)

بۆ پشتڕاستکردنەوەی کۆگای ڕاستەقینە:

1. بڕۆ بۆ **کۆگا** > **ژماردنی کۆگا**
2. کلیک لەسەر **دروستکردنی ژماردن** بکە
3. هەڵبژێرە: کۆگا، جۆری ژماردن
4. بڕە ژمێردراوەکان بنووسە بۆ هەر بەرهەمێک
5. سیستەم جیاوازی پیشان دەدات
6. کلیک لەسەر **ناردن** بکە بۆ جێبەجێکردنی ڕاستکردنەوەکان

## ژمێریاری

### پلانی هەژمارەکان (Chart of Accounts)

بینین و بەڕێوەبردنی هەژمارەکانت:

1. بڕۆ بۆ **ژمێریاری** > **هەژمارەکان**
2. هەژمارەکان بە جۆر ڕێکخراون ببینە: سامان، قەرزەکان، مافی خاوەنداران، داهات، خەرجییەکان
3. بۆ دروستکردنی هەژمار:
   - کلیک لەسەر **دروستکردنی هەژمار** بکە
   - کۆد و ناوی هەژمار بنووسە
   - جۆری هەژمار هەڵبژێرە
   - هەژماری باوک (Parent) هەڵبژێرە (بۆ هەژمارە لاوەکان)
   - کلیک لەسەر **پاشەکەوت** بکە

### تۆمارەکانی ڕۆژنامە (Journal Entries)

بۆ تۆمارە ژمێریارییە دەستیەکان:

1. بڕۆ بۆ **ژمێریاری** > **تۆمارەکانی ڕۆژنامە**
2. کلیک لەسەر **دروستکردنی تۆمار** بکە
3. بنووسە: بەرواری تۆمار، ژمارەی سەرچاوە، وەسف
4. دێڕەکان زیاد بکە: هەژمار هەڵبژێرە، قەرزدار (Debit) یان قەرزخواز (Credit) بنووسە
5. **کۆی قەرزدار دەبێت یەکسان بێت بە کۆی قەرزخواز**
6. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی خەرجی

1. بڕۆ بۆ **ژمێریاری** > **خەرجییەکان**
2. کلیک لەسەر **دروستکردنی خەرجی** بکە
3. پڕ بکەرەوە: بەروار، جۆری خەرجی، بڕ، وەسف
4. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی باج

#### ڕێکخستنی کۆدەکانی باج
1. بڕۆ بۆ **ژمێریاری** > **کۆدەکانی باج**
2. کلیک لەسەر **دروستکردنی کۆدی باج** بکە
3. بنووسە: ناو، ڕێژە، جۆری باج
4. کلیک لەسەر **پاشەکەوت** بکە

## کارمەندان (HR)

### بەڕێوەبردنی کارمەند

#### زیادکردنی کارمەند

1. بڕۆ بۆ **کارمەندان** > **کارمەندەکان**
2. کلیک لەسەر **دروستکردنی کارمەند** بکە
3. **زانیاری کەسی** پڕ بکەرەوە:
   - ناوی تەواو
   - کۆدی کارمەند
   - بەرواری لەدایکبوون
   - ڕەگەز
   - ژمارەی ناسنامە
   - زانیاری پەیوەندی
4. **وردەکاری کارکردن** پڕ بکەرەوە:
   - بەش
   - پۆست
   - بەرواری دەستپێکردن
   - جۆری کارکردن
   - خشتەی کار
5. **قەرەبوو** پڕ بکەرەوە:
   - مووچەی بنەڕەت
   - وردەکاری هەژماری بانکی
6. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی ئامادەبوون

1. بڕۆ بۆ **کارمەندان** > **ئامادەبوون**
2. کلیک لەسەر **دروستکردنی ئامادەبوون** بکە
3. کارمەند هەڵبژێرە
4. بەروار و کاتەکان بنووسە
5. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی مۆڵەت

#### داواکاری مۆڵەت (وەک کارمەند)

1. بڕۆ بۆ **کارمەندان** > **داواکاری مۆڵەت**
2. کلیک لەسەر **دروستکردنی داواکاری** بکە
3. هەڵبژێرە: جۆری مۆڵەت، بەرواری دەستپێکردن، بەرواری کۆتایی، هۆکار
4. کلیک لەسەر **ناردن** بکە

#### پەسەندکردنی مۆڵەت (وەک بەڕێوەبەر)

1. بڕۆ بۆ **کارمەندان** > **داواکاری مۆڵەت**
2. داواکاریە هەڵواسراوەکان بدۆزەرەوە
3. کلیک لەسەر داواکاری بکە بۆ پێداچوونەوە
4. کلیک لەسەر **پەسەندکردن** یان **ڕەتکردنەوە** بکە

### ئامادەکردنی مووچە

#### ئامادەکردنی مووچەی مانگانە

1. بڕۆ بۆ **کارمەندان** > **ماوەکانی مووچە**
2. کلیک لەسەر **دروستکردنی ماوە** بکە
3. مانگ و ساڵ هەڵبژێرە
4. کلیک لەسەر **دروستکردنی لیستی مووچە** بکە
5. لیستە دروستکراوەکان ببینەوە
6. هەر گۆڕانکارییەک بکە
7. کلیک لەسەر **پەسەندکردنی مووچە** بکە
8. کلیک لەسەر **تۆمارکردنی پارەدان** بکە

## ڕاپۆرتەکان

دەستگەیشتن بە ڕاپۆرتەکان لە **ڕاپۆرتەکان** لە مێنووی سەرەکی.

### ڕاپۆرتە داراییەکان

#### تەرازووی گشتی (Balance Sheet)
سامان، قەرزەکان، و مافی خاوەنداران لە کاتێکی دیاریکراودا پیشان دەدات

#### پوختەی داهات و خەرجی (Income Statement)
داهات و خەرجییەکان بۆ ماوەیەک پیشان دەدات

#### تەرازووی تاقیکردنەوە (Trial Balance)
باڵانسی هەموو هەژمارەکان لیست دەکات

#### ڕاپۆرتی لێشاوی پارە (Cash Flow)
جوڵەی پارە بە چالاکی پیشان دەدات

### ڕاپۆرتەکانی فرۆشتن

- **ڕاپۆرتی فرۆشتن** - پوختەی هەموو فرۆشتن
- **فرۆشتن بە بەرهەم** - کارایی بە بەرهەم
- **باشترین کڕیاران** - ڕیزبەندی کڕیاران بە کڕین
- **باڵانسی کڕیاران** - شیکاری تەمەنی قەرزدارەکان

### ڕاپۆرتەکانی کۆگا

- **ڕاپۆرتی جوڵەی کۆگا** - هەموو مامەڵەکانی کۆگا
- **نرخاندنی کۆگا** - بەهای کۆگای ئێستا

### هەناردەی ڕاپۆرتەکان

هەموو ڕاپۆرتەکان هەناردە پشتگیری دەکەن:
1. ڕاپۆرت دروست بکە
2. کلیک لەسەر **هەناردە** بکە
3. جۆر هەڵبژێرە (PDF، Excel، CSV)
4. فایل بە شێوەی خۆکار دادەبەزێت

## ڕێکخستنەکانی سیستەم

### بەڕێوەبردنی بەکارهێنەر

#### دروستکردنی بەکارهێنەر

1. بڕۆ بۆ **ڕێکخستنەکان** > **بەکارهێنەران**
2. کلیک لەسەر **دروستکردنی بەکارهێنەر** بکە
3. بنووسە: ناو، ئیمەیڵ، وشەی نهێنی، ڕۆڵ
4. کلیک لەسەر **پاشەکەوت** بکە

### بەڕێوەبردنی ڕۆڵ

1. بڕۆ بۆ **ڕێکخستنەکان** > **ڕۆڵەکان**
2. ڕۆڵە بەردەستەکان و مۆڵەتەکانیان ببینە

### بەڕێوەبردنی مۆڵەت

1. بڕۆ بۆ **ڕێکخستنەکان** > **مۆڵەتەکان**
2. هەموو مۆڵەتە بەردەستەکان ببینە
3. مۆڵەتەکان بە بەش ڕێکخراون

## چارەسەرکردنی کێشەکان

### کێشە باوەکان

#### ناتوانم بچمە ژوورەوە
- دڵنیابە ئیمەیڵ و وشەی نهێنی ڕاستن
- بپشکنە هەژمارەکەت چالاکە
- پەیوەندی بە سەرپەرشتیار بکە ئەگەر قفڵکرابوو

#### POS کار ناکات
- دڵنیابە شەفتێکی کراوەت هەیە
- بپشکنە ڕۆڵەکەت دەستگەیشتنی هەیە بە POS
- پەڕەکە نوێ بکەرەوە

#### بەرهەمەکان نییە
- بپشکنە بەرهەم وەک "چالاک" دانراوە
- بپشکنە بەرهەم کۆگای هەیە
- هەڵبژاردەی کۆگا بپشکنە
