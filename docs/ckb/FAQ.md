# پرسیارە باوەکان (FAQ) بۆ GoPOS

---

## پرسیارە گشتییەکان

### GoPOS چییە؟
GoPOS سیستەمێکی تەواوی خاڵی فرۆشتن (POS) و بەڕێوەبردنی بازرگانییە کە بۆ کڕین و فرۆشتن دیزاین کراوە. خاڵی فرۆشتن، بەڕێوەبردنی کۆگا، ژمێریاری، و بەشی کارمەندان (HR) لەخۆ دەگرێت.

### چ وێبگەڕانەیەک پشتگیری دەکرێت؟
GoPOS باشتر کار دەکات لەسەر:
- Google Chrome (پێشنیار دەکرێت)
- Mozilla Firefox
- Microsoft Edge
- Safari

### دەتوانم GoPOS لەسەر مۆبایل بەکار بهێنم؟
بەڵێ، GoPOS بەرپرسیارە (Responsive) و لەسەر تابلێت کار دەکات. بۆ کارەکانی خاڵی فرۆشتن، پێشنیار دەکەین تابلێت لە شێوازی ئاسۆیی (Landscape) یان کۆمپیوتەری دێسکتۆپ/لاپتۆپ بەکار بهێنیت.

### GoPOS چ زمانانەیەک پشتگیری دەکات؟
- ئینگلیزی
- عەرەبی
- کوردی سۆرانی

دەتوانیت زمانەکەت بگۆڕیت لە ڕێکخستنەکانی پڕۆفایلەکەت.

## هەژمار و چوونەژوورەوە

### وشەی نهێنیم بیرچووەتەوە. چی بکەم؟
کلیک لەسەر "وشەی نهێنیم بیرچووەتەوە" بکە لە پەڕەی چوونەژوورەوە، ئیمەیڵەکەت بنووسە، و ڕێنماییەکان جێبەجێ بکە کە بۆ ئیمەیڵەکەت دەنێردرێت.

### بۆچی ناتوانم چەند تایبەتمەندییەک ببینم؟
دەسەڵاتەکانت بە ڕۆڵەکەت دیاری دەکرێت. پەیوەندی بە بەڕێوەبەر بکە ئەگەر مۆڵەتی زیاترت دەوێت.

### چۆن وشەی نهێنیم بگۆڕم؟
1. کلیک لەسەر ناوەکەت بکە (گۆشەی سەرەوەی چەپ/ڕاست)
2. **پڕۆفایل** هەڵبژێرە
3. بڕۆ بۆ بەشی **وشەی نهێنی**
4. وشەی نهێنی ئێستا و نوێ بنووسە
5. کلیک لەسەر **پاشەکەوتکردن** بکە

## کارەکانی خاڵی فرۆشتن (POS)

### بۆچی ناتوانم POS بەکار بهێنم؟
دڵنیابە کە:
1. مۆڵەتی دەستڕاگەیشتنت هەیە بۆ POS (ڕۆڵی کاشێر، بەڕێوەبەر، یان سەرپەرشتیار)
2. شەفتێکی کراوەت هەیە (بڕۆ بۆ **POS** > **بەڕێوەبردنی شەفت** > **کردنەوەی شەفت**)

### چۆن گەڕاندنەوە پرۆسێس بکەم؟
1. بڕۆ بۆ **فرۆشتن** > **گەڕاندنەوەی فرۆشتن**
2. کلیک لەسەر **دروستکردنی گەڕاندنەوە** بکە
3. فرۆشتنی ڕەسەن هەڵبژێرە
4. ئایتمەکان هەڵبژێرە کە دەگەڕێنرێنەوە
5. گەڕاندنەوەکە تەواو بکە

### دەتوانم پارەدان دابەش بکەم لەنێوان نەقد و کارت؟
بەڵێ! لە کاتی پارەدان:
1. کلیک لەسەر **دابەشکردنی پارەدان** بکە
2. بڕی نەقد بنووسە
3. پارەدانی کارت زیاد بکە بۆ ماوەکە
4. فرۆشتنەکە تەواو بکە

### چۆن فرۆشتنێک هەڵبگرم بۆ دواتر (Hold)؟
1. بەرهەمەکان زیاد بکە بۆ سەبەتە
2. کلیک لەسەر **هەڵگرتنی فرۆشتن** بکە
3. تێبینی بنووسە (وەک ناوی کڕیار)
4. بۆ بیرهێنانەوەی: کلیک لەسەر **فرۆشتنە هەڵگیراوەکان** بکە و هەڵیبژێرە

### بۆچی هەندێک بەرهەم لە POS نییە؟
بەرهەمەکان ڕەنگە نەبینرێن ئەگەر:
- وەک ناچالاک دانراون
- بڕی سفریان هەیە (ئەگەر بەدواداچوونی کۆگا چالاک بێت)
- بۆ کۆگاکەت دیاری نەکراون

### چۆن داشکاندن جێبەجێ بکەم؟
لە کاتی پارەدان:
1. کلیک لەسەر **داشکاندن** بکە
2. ڕێژەی سەدی (%) یان بڕی جێگیر بنووسە
3. داشکاندنەکە لەسەر کۆی گشتی جێبەجێ دەکرێت

### ڕاپۆرتی Z چییە؟
ڕاپۆرتی Z (یان ڕاپۆرتی شەفت) هەموو مامەڵەکان لە ماوەی شەفتدا کورت دەکاتەوە، لەوانە:
- کۆی فرۆشتن
- وردەکاری شێوازەکانی پارەدان
- جوڵەی پارە (هاتوو/دەرچوو)
- جیاوازی (پارەی چاوەڕوانکراو بەرامبەر پارەی ژمێردراو)

## کۆگا (Inventory)

### چۆن بڕی بەرهەم بپشکنم؟
1. بڕۆ بۆ **کۆگا** > **بەرهەمەکان**
2. بەرهەمەکەت بدۆزەرەوە
3. بڕی کۆگا لە لیستەکە یان وردەکاری بەرهەم دەبینرێت

### چۆن بەرهەمی نوێ زیاد بکەم؟
1. بڕۆ بۆ **کۆگا** > **بەرهەمەکان**
2. کلیک لەسەر **دروستکردنی بەرهەم** بکە
3. ناو، SKU، نرخ، هتد پڕ بکەرەوە
4. کلیک لەسەر **پاشەکەوتکردن** بکە

### جیاوازی چییە لەنێوان بەدواداچوونی وەجبە (Batch) و ژمارەی زنجیرەیی (Serial)؟

**بەدواداچوونی وەجبە (Batch):**
- بەرهەمەکان بە وەجبەی بەرهەمهێنان کۆدەکاتەوە
- بەرواری بەسەرچوون بەدوادەچێت
- باشە بۆ خۆراک، دەرمان، و جوانکاری

**بەدواداچوونی ژمارەی زنجیرەیی (Serial):**
- هەر یەکەیەک ژمارەی زنجیرەیی تایبەتی هەیە
- دانە تاکەکان بەدوادەچێت
- باشە بۆ ئەلیکترۆنیات و ئامێرەکان

### چۆن کۆگا بگوازمەوە لەنێوان کۆگاکان؟
1. بڕۆ بۆ **کۆگا** > **گواستنەوەی کۆگا**
2. کلیک لەسەر **دروستکردنی گواستنەوە** بکە
3. کۆگای سەرچاوە و مەبەست هەڵبژێرە
4. بەرهەم و بڕ زیاد بکە
5. بۆ پەسەندکردن پێشکەش بکە
6. دوای پەسەندکردن، گواستنەوەکە وەربگرە

### چی بکەم ئەگەر ژمارەی کۆگا یەکسان نەبوو؟
1. بڕۆ بۆ **کۆگا** > **ژماردنی کۆگا**
2. ژماردنی کۆگا دروست بکە
3. بڕە ڕاستەقینەکان بنووسە
4. جیاوازییەکان بپشکنە
5. بۆ ڕاستکردنەوەی کۆگا بە شێوەی خۆکار جێبەجێی بکە

## ژمێریاری

### چۆن خەرجی تۆمار بکەم؟
1. بڕۆ بۆ **ژمێریاری** > **خەرجییەکان**
2. کلیک لەسەر **دروستکردن** بکە
3. جۆری خەرجی هەڵبژێرە
4. بڕ و بەروار بنووسە
5. پاشەکەوت بکە

### چۆن تۆماری ڕۆژنامە (Journal Entry) دروست بکەم؟
1. بڕۆ بۆ **ژمێریاری** > **تۆمارەکانی ڕۆژنامە**
2. کلیک لەسەر **دروستکردنی تۆمار** بکە
3. دێڕەکانی قەرزدار (Debit) و قەرزخواز (Credit) زیاد بکە
4. **گرنگ:** کۆی قەرزدار دەبێت یەکسان بێت بە کۆی قەرزخواز
5. پاشەکەوت بکە

### لەکوێ دەتوانم ڕاپۆرتە داراییەکان ببینم؟
بڕۆ بۆ **ڕاپۆرتەکان** و هەڵبژێرە:
- **تەرازووی گشتی** (Balance Sheet) - سامان، قەرزەکان، و مافی خاوەنداران
- **پوختەی داهات و خەرجی** (Income Statement) - داهات و خەرجییەکان
- **تەرازووی پێداچوونەوە** (Trial Balance) - باڵانسی هەموو هەژمارەکان
- **لێشاوی پارە** (Cash Flow) - جوڵەی نەقدینە

## کارمەندان (HR)

### چۆن کارمەندی نوێ زیاد بکەم؟
1. بڕۆ بۆ **کارمەندان** > **کارمەندەکان**
2. کلیک لەسەر **دروستکردنی کارمەند** بکە
3. زانیاری کەسی و دامەزراندن پڕ بکەرەوە
4. پاشەکەوت بکە

### چۆن ئامادەبوون تۆمار بکەم؟
1. بڕۆ بۆ **کارمەندان** > **ئامادەبوون**
2. کلیک لەسەر **دروستکردنی ئامادەبوون** بکە
3. کارمەند هەڵبژێرە و کاتەکان بنووسە
4. پاشەکەوت بکە

### چۆن مووچە ئامادە بکەم؟
1. بڕۆ بۆ **کارمەندان** > **ماوەکانی مووچە**
2. کلیک لەسەر **دروستکردنی ماوە** بکە
3. مانگ/ساڵ هەڵبژێرە
4. کلیک لەسەر **ئامادەکردنی مووچە** بکە
5. پێداچوونەوە بکە و پەسەند بکە
6. پارەدان تۆمار بکە

## چارەسەرکردنی کێشەکان

### پەڕەکە بار نابێت
1. پەڕەکە نوێ بکەرەوە (F5 یان Ctrl+R)
2. کاشی (Cache) وێبگەڕ پاک بکەرەوە
3. وێبگەڕێکی تر تاقی بکەرەوە
4. پەیوەندی ئینتەرنێت بپشکنە

### گۆڕانکارییەکانم پاشەکەوت نابێت
1. هەڵەکانی پشتڕاستکردنەوە (Validation Errors) بپشکنە (دەقی سور)
2. دڵنیابە هەموو خانە پێویستەکان پڕکراونەتەوە
3. نوێکردنەوە و دووبارە بنووسە

### هەڵەم کرد لە مامەڵەیەکدا
- بۆ فرۆشتن: گەڕاندنەوەی فرۆشتن دروست بکە
- بۆ کڕین: گەڕاندنەوەی کڕین دروست بکە
- بۆ تۆمارەکانی ڕۆژنامە: تۆماری پێچەوانە (Reversal) دروست بکە
- پەیوەندی بە بەڕێوەبەر یان سەرپەرشتیار بکە بۆ یارمەتی

## یارمەتی زیاتر

ئەگەر وەڵامی پرسیارەکەت لێرە نەدۆزییتەوە:

1. **ڕێنمایی بەکارهێنەر بخوێنەرەوە:** سەیری `USER-GUIDE.md` بکە بۆ ڕێنمایی ورد
2. **لە بەڕێوەبەرەکەت بپرسە:** بۆ پرسیارەکانی پڕۆسە و سیاسەت
3. **پەیوەندی بە پشتگیری تەکنیکی بکە:** بۆ کێشە تەکنیکییەکان
4. **هەڵە ڕاپۆرت بکە:** کێشەکە و هەنگاوەکانی دووبارەکردنەوە تۆمار بکە
